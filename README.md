# FlaskYTDl
Youtube mp3/mp4 downloader built with Python/Flask backend, Vanilla HTML/CSS/JS frontend.
Uses yt-dlp.

## Installation
```console
$ git clone https://github.com/tnk-rin/FlaskYTDl.git
$ cd FlaskYTDl
$ pip install -r requirements.txt
$ flask run
```
